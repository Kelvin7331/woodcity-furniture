# WOODCITY — Crafted Furniture

A luxury, high-end furniture showcase website featuring a minimalist aesthetic, immersive animations, and a seamless user experience.

## ✨ Features
- **Immersive UI:** Custom interactive cursor and subtle noise overlay for a premium feel.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Elements:** Smooth scroll behavior, marquee tracks, and reveal animations.
- **Functional Forms:** Includes a detailed contact section for project inquiries.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables & Keyframe Animations)
- **Typography:** Cormorant Garamond (Serif) and Jost (Sans-serif)
- **Icons:** Font Awesome 6
